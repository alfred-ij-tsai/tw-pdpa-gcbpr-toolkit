03為什麼對照 Global CBPR

可以把它想成隱私護照。通過一次驗證，效力在全體成員經濟體之間相互承認，不必為每一個市場重做一次合規論證。

臺灣是 Global CBPR Forum 的創始成員，國家發展委員會擔任協調角色，財團法人資訊工業策進會為我國的當責機構（Accountability Agent）。這意味著臺灣企業要取得跨境資料流通的國際互信，Global CBPR 是現成且已就位的制度路徑。

選擇 Global CBPR 作為對照基準，還有一個務實理由：Annex B 的 Enforcement Map 是公開文件，且已有中英對照版本。這使得本工具箱得以完整引用其問題全文，並以開放授權釋出，任何人都能自由取用、修改與再散布。

本工具箱的核心始終是臺灣個資法的日常法令遵循。GCBPR 欄位是附加的對照參考，供有跨境需求或正在準備驗證的組織使用，不影響本國法義務的判斷。

04版本、範圍與限制

工具箱版本
v1.1（2026 年 7 月）。本版為網頁互動版首次發布，內容依現行有效條文更正自 v1.0 之 Word 版本，更正項目詳見 GitHub 文件庫之版本說明。
個人資料保護法
現行有效條文。114.11.11 修正公布之條文，施行日期由行政院定之，尚未施行；涉及該次修正處均另行標示。
施行細則
105 年 3 月 2 日修正發布
行政院要點
行政院及所屬各機關落實個人資料保護聯繫作業要點（114.01.07 修正）
對照基準
Global CBPR System Program Requirements Enforcement Map for Chinese Taipei（Annex B）
適用對象
非公務機關。公務機關之安全維護義務依個資法第 18 條，並另有政府資訊公開法、檔案法、行政程序法等特別規定，不在涵蓋範圍。
收錄範圍
個資法及其授權訂定之法規命令。各業別之勞動、醫療、金融、電信等專業法規不在收錄範圍，惟醫療法第 68 條、第 71 條因直接牽動當事人權利之履行方式，例外納入 04 表。
期限計算
各底稿所列日期均為依日數或時數計算之原始到期日，未計入例假日順延。實際末日請依行政程序法第 48 條並核對當年度辦公日曆表後確認。

本工具箱為法令遵循參考資料，不構成法律意見。所引法規應以 全國法規資料庫現行有效版本為準。 產出之底稿為工作底稿，具體個案請諮詢法律專業人士。
